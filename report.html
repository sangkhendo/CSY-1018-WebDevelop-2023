Report

Introduction:

This report discusses the experience of learning the basics of web development, including coding, design, and the debugging process. The website created for this assignment and the design decisions made during the development process are reflected upon.

Learning the basics of web development:

The fundamentals of web development, such as HTML and CSS, were learned during this module.HTML was used to create web pages while CSS was used to style them. Html is being used to add interactivity to the web pages, providing a solid foundation for future endeavors.

Coding, design, and debugging process:

The website development process was not without its challenges. Positioning elements using CSS and debugging code proved to be difficult, but overcoming these challenges through trial and error resulted in a functioning website. Testing and debugging were emphasized to ensure that the code worked as intended.

Design Decision:

Many design considerations were taken during the development process, such as selecting a color palette, selecting appropriate fonts, and settling on the website layout. A basic design was eventually chosen, with a black and red color scheme and a clean, modern typography. A responsive design was also used to guarantee that the website is optimized for different devices.

Reflective discussion of module experience:

This module was found to be valuable due to its comprehensive introduction to web development and the practical application of skills in a real-world context. Learning about html and css and its use in adding interactivity to web pages was particularly enjoyable. Design decisions, such as a minimalist design with a black and red color scheme, clean modern font and a responsive design that looks good on various devices, were made during the development process.

Conclusion:

In conclusion, this module provided a valuable introduction to web development and the skills necessary to create a basic website.The challenges faced during the development process emphasized the importance of testing and debugging.The reflection on design decisions made highlights the importance of design in creating a visually appealing and functional website.
